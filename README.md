# Project 8 – ABR OAE Hearing Analysis

Feature-based analysis of auditory brainstem responses (ABR) and otoacoustic emissions (OAE)
for hearing-status classification and stimulus-level prediction.

## Goals
- Classify normal-hearing vs hearing-impaired listeners
- Predict stimulus level / loudness-related targets
- Compare time-domain, frequency-domain, and reliability-based features

## Structure
- `data/` raw and processed data
- `notebooks/` exploration and experiments
- `src/` reusable code
- `results/` figures, tables, outputs
